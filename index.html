Welcome to Morpheus!

Morpheus is an exciting new linear algebra package. It currently contains a dense matrix and dense vector class, but we hope to expand its features as it grows in popularity.

How does Morpheus compare to other linear algebra packages?

It has way fewer features, but what we lack in features, we make up for in bugs.

Is Morpheus greater than sliced bread?

That's not even a fair question. Morpheus is the best thing that's ever happened to math since the discovery of 0. Speaking of which, the history of the number 0 is super-cool and can be found here: https://en.wikipedia.org/wiki/0_%28number%29#History

Is this entire project just a Doxygen exercise?

No. It is also an exercise on using code coverage tools such as gcov.
